# Conference Website

Website for organising remote conferences. Integrates with the WhereBy API.

**Frontend**
- React (Next.js using TypeScript)
- Material UI
- Zustand for State Management
- React Query
- Moment

**Backend**
- .Net 6 API
- Entity Framework
- Unit of Work pattern
- Redis
- Auth0
- RestSharp
- AutoMapper 

Backend code is in a separate private repository
